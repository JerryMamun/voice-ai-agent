# Voice AI Agent

## Stack
- FastAPI (Backend)
- Render (Hosting)
- Deepgram (STT)
- Groq (LLM)
- ElevenLabs (TTS)
- Vercel (Frontend)

## Setup

1. Add API keys in Render
2. Deploy backend
3. Update frontend API URL
4. Deploy frontend

## Endpoints

POST /voice/chat (audio)
POST /voice/text (text)
